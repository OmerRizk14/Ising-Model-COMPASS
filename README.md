# 2D Ising Model Simulation

A computational physics project built in Python to simulate and visualize the *2D Ising Model* using Monte Carlo algorithms.
This simulation helps analyze thermodynamic behaviors, phase transitions, energy states, and magnetization in ferromagnetic systems.

## 🚀 Features
* Simulates spin configurations on a 2D grid.
* Computes and plots *Average Magnetization ($M$) vs Timesteps*.
* Computes and plots *Total Energy ($E$) vs Timesteps*.
* Allows user inputs for temperature, steps, and initial spin bias.
